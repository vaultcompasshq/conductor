# @vaultcompass/conductor-core

Core libraries for [Conductor](https://github.com/vaultcompasshq/conductor): prompt coaching, drift scoring, enforcement gate, corrections, session brief, memory index, and setup diagnostics.

## Install

```bash
npm install @vaultcompass/conductor-core
```

Typically consumed via `@vaultcompass/conductor-cli` or `@vaultcompass/conductor-skill`. Use this package when embedding Conductor in Node tooling.

## Exports

Coaching, drift, gate, contract store, init, doctor, report, rules audit, spec bridge, history, and correction utilities. See `dist/index.d.ts` after install.
